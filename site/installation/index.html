Installation

Please take the steps below to install OPBNOS API.

Prerequisites

  • Python version 3.7+
  • pip version 21.2.4+
  • Download opbnos-api-0.1.tar.gz

Install package

pip install opbnos-api-0.1.tar.gz

If above command does not work, try using pip3:

pip3 install opbnos-api-0.1.tar.gz
